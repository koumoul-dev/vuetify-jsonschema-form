Density

Many vuetify components accept a density prop with possible values default, comfortable and compact.

Vjsf also accepts this parameter as an option. It is applied to all relevant vuetify components and is also used to modify title sizes, margins, etc.

Default

The default density is very low, titles are big and there are large white spaces.

Comfortable

The comfortable density is a well balanced middleground.

Compact

The compact density is very high, titles and margins are small.
